### Условие

Всеки от процесите $P$ и $Q$ изпълнява поредица от две инструкции:

| $P$   | $Q$   |
|-------|-------|
| $p_1$ | $q_1$ |
| $p_2$ | $q_2$ |

Осигурете чрез семафори синхронизация на $P$ и $Q$ така, че инструкция $p_1$ да се изпълни преди
$q_2$, а $q_1$ да се изпълни преди $p_2$.

### Решение


| Initialize     |
| -------------- |
| Semaphore s,t |
| s.init(0)     |
| t.init(0)     |

| $P$        | $Q$        |
| ---------- | ---------- |
| $p_1$      | $q_1$      |
| s.signal() | s.wait()   |
| t.wait()   | t.signal() |
| $p_2$      | $q_2$      |
